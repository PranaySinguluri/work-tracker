Google Calendar Setup

Connect your Google Calendar to sync events. You'll need a Google Cloud project with the Calendar API enabled.

How to set up
  1. Go to Google Cloud Console
  2. Create a project → Enable Google Calendar API
  3. Create OAuth 2.0 credentials (Web App)
  4. Add your GitHub Pages URL to Authorized Origins
  5. Copy Client ID and API Key here
